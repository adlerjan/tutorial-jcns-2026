# HermesPy Tutorial

Accompanying source code to the tutorial titled *Joint evaluation of radar and communication applications on the physical layer: A deep-dive into HermesPy* presented at the 6th IEEEInternational Symposium on
Joint Communications & Sensing in In Ponte di Legno, Italy.

## Requirements

Python 3.12 and an installation of [HermesPy](https://github.com/Barkhausen-Institut/hermespy) from source. Refer to the [installation instructions from source](https://hermespy.org/installation.html#install-from-source) for step-by-step instructions.
